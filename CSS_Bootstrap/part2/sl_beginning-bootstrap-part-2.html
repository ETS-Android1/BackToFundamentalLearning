The Grid System:

Bootstrap Grid System allows up to 12 columns across the page.

You can use each of them individually or merge them together for wider columns.

You can use all combinations of values summing up to 12.

You can use 12 columns each of width 1, or use 4 columns each of width 3 or any other combination.

Grid Classes:

The Bootstrap grid system has four classes which can be combined to make more flexible layouts:

  • xs – For Mobile Phones.
  • sm – For Tablets/Phablets
  • md – For Small-sized Desktops/Laptops
  • lg – For Larger-sized Desktops/Laptops
